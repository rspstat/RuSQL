## 2p. 프로젝트 기능

**No. 1 — RDBMS 시스템**
관계형 데이터베이스에서 SQL 쿼리문 작성을 통해 데이터를 생성·조회·수정·삭제 등을 할 수 있는 데이터베이스 엔진 기능

**No. 2 — 네트워크 서버**
TCP 기반 서버를 생성하고, 인가된 다중 사용자가 데이터베이스에 접근하여 동시에 데이터를 조작할 수 있도록 하는 실시간 시스템 기능

**No. 3 — GUI**
GUI를 기반으로 SQL 쿼리문 없이 화면에서 테이블을 생성 및 조작하고, 관계를 설정할 수 있는 시각적 편집기 기능

**No. 4 — AI**
인공지능의 MCP 기술을 기반으로 비전문 사용자도 SQL 쿼리 작성 없이 자연어를 통해 데이터를 조작할 수 있도록 하는 기능

---

## 3p. 프로젝트 평가 계획

4가지 핵심 기능을 안전하고 완전하게 사용 가능한 형태를 완성 기준으로 삼고 평가한다.

**RDBMS :**
- 실제 SQL 쿼리문을 작성하고 실행했을 때 정확한 결과가 나오는가
- 데이터 변경 중 오류가 발생했을 때 데이터가 안전하게 복구되는가

**서버 :**
- 여러 사용자가 동시에 접속해서 같은 데이터베이스를 다룰 수 있는가
- 암호 등을 통해 인가받은 사용자만이 데이터베이스에 접근할 수 있는가

**GUI :**
- SQL 쿼리 없이도 시각적으로 테이블을 만들고 데이터를 조작할 수 있는가
- 테이블 간의 관계(FK)를 시각적으로 직접 설정하고 확인할 수 있는가

**AI :**
- 자연어로 질의 쿼리를 작성했을 때 원하는 쿼리문으로 작성해주는가
- AI로 생성한 자연어 기반 쿼리문이 올바른 로직을 수행하는가
