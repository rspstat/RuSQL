🔧 실제적인 행동/처리 관련
Coalescing : 여러 요소를 하나로 합치는 것 (통합)
Compacting : 공간을 덜 차지하게 압축하는 것
Generating : 무언가를 만들어내는 것 (생성)
Processing : 데이터를 처리하거나 절차를 수행하는 것
Actioning : 계획을 실제 행동으로 옮기는 것
Accomplishing : 목표나 작업을 완수하는 것
Creating : 새로운 무언가를 창조하는 것
Doing : 실제로 무언가를 수행하는 일반적인 행위
Working : 작업하거나 일을 하는 행위
Forming : 형태를 만들거나 구조화하는 것
Actualizing : 아이디어나 가능성을 현실로 구현하는 것
Transmuting : 어떤 것을 완전히 다른 것으로 바꾸는 것 (변형)
Churning : 반복적으로 움직이거나 생산하는 것 (돌리고 만드는 것)
Baking : 요리 외에도 무언가를 ‘굽듯이’ 천천히 완성시키는 것
Shimmying : 흔들거리며 움직이는 것 (가볍고 유연한 행동)
Schlepping : 무거운 것을 낑낑대며 나르는 것
Hustling : 바쁘고 열정적으로 움직이는 것
Mustering : 에너지나 용기를 모으는 것 (기운 내기)
Tinkering : 사소한 부분을 만지작거리며 개선하려고 하는 것
Puttering : 목적 없이 이것저것 하며 시간을 보내는 것

💡 아이디어, 생각 관련
Thinking : 사고하거나 아이디어를 떠올리는 일반적인 행위
Pondering : 깊이 고민하거나 심사숙고하는 것
Ideating : 아이디어를 떠올리거나 구상하는 과정
Brewing : 아이디어나 계획이 서서히 떠오르는 중인 상태
Percolating : 생각이나 아이디어가 점점 명확해지는 상태
Stewing : 걱정이나 아이디어를 계속 곱씹는 것
Noodling : 가볍게 생각하거나 실험적으로 머릿속에서 만지는 것
Germinating : 생각이나 아이디어가 씨앗처럼 싹트는 것
Hatching : 새로운 계획이나 아이디어를 준비하거나 떠올리는 것

🔍 탐색/분석 관련
Deciphering : 암호나 복잡한 정보를 해독하는 것
Spelunking : 동굴 탐험 (비유적으로 미지의 영역을 탐색)
Perusing : 꼼꼼히 정독하거나 검토하는 것
Inferring : 단서를 통해 추론하거나 결론을 도출하는 것
Crunching : 데이터를 계산하거나 분석하는 것
Sussing : 상황을 파악하거나 진상을 캐내는 것
Unravelling : 얽힌 문제나 미스터리를 푸는 것
✨ 창의적 상상/마법적 상징
Conjuring : 마술처럼 아이디어나 무언가를 떠올리는 것
Enchanting : 매혹하거나 마법을 부리는 듯한 매력 발산
Manifesting : 생각이나 의도를 현실로 끌어오는 것
Channelling : 어떤 에너지나 아이디어를 특정 방향으로 흐르게 하는 것
Divining : 직감이나 예지로 정보를 얻으려는 행위
Vibing : 분위기나 감정을 타거나 공유하는 것
Combobulating : 혼란을 정리하고 질서를 회복하는 것

🌀 혼란/방황/미지
Discombobulating : 혼란스럽게 하거나 어지럽히는 것
Puzzling : 혼란스러운 상황을 마주하거나 이해하려 애쓰는 것
Wandering : 목적 없이 이리저리 떠도는 것
Meandering : 방향 없이 돌아다니는 것 (정신적 또는 물리적)
Pontificating : 아는 체하며 거들먹거리는 식의 설명
Elucidating : 어렵거나 애매한 것을 명확하게 설명하는 것
Unfurling : 접힌 것을 펼치듯 생각이나 가능성을 확장하는 것
Smoothing : 거친 것을 매끄럽게 만들거나 조정하는 것
